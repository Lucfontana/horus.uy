# horus.uy
Proyecto final del programa "Jóvenes Creativos"

## Problemática a resolver
Horus.uy es una tienda de ropa ubicada en la ciudad de Paysandú, Uruguay que busca brindar a sus clientes ropa de calidad a la moda, con un precio más accesible que en otras tiendas locales.

Con un público cada vez más en aumento, y con un método de catálogo actual basado meramente en publicaciones en el Instagram oficial, surge la necesidad de cambiar la forma de mostrar el catálogo de ropa disponible. Se busca hacerlo de una manera más fácil y accesible para los clientes, por lo que el desarrollo de un E-Commerce se vuelve esencial para completar esta tarea.

## Público objetivo
Adolescentes y adultos de cualquier género

## Productos/Servicios principales

### Ropa de todo tipo:
- Mixto: Pantalones, joggings, sudaderas, buzos, calzados.
- Para hombres: Camisetas (formales e informales).
- Para mujeres: Vestidos, calzados.
